Features:

Program can browse, create and delete directories. That's all.

Installing: 

1. Create a separate foolder for the file browser in your Ampps www folder.
2. Copy all the files form the github repository to the created folder. 
3. Type "localhost/name of your folder/" in your prefered web browser's search bar.

Use:

Folders already present in the file browser's main folder will be displayed under the search bar. 
Create new folders by typing their name in the search bar.
Delete folders by clicking the red cross near their names.
